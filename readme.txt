Adds a "Personal Pickup" shipping method with store selection for WooCommerce.

== Description ==

This plugin adds a custom "Personal Pickup" shipping method to WooCommerce that allows customers to select from multiple store locations at checkout.

= Features =

* Custom "Personal Pickup" shipping method
* Manage multiple store locations with address, hours, phone, and email
* Store selection dropdown at checkout
* Customizable field labels
* Mobile responsive design
* Display store information on Thank You page and emails

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/wc-pickup-store`, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to WooCommerce → Pickup Store to add your store locations.
4. Go to WooCommerce → Settings → Shipping → Shipping Zones and add the "Personal Pickup" shipping method to your zone.

== Frequently Asked Questions ==

= How do I add stores? =

Go to WooCommerce → Pickup Store → Add New Store and fill in the store details.

= How do I customize the labels? =

Go to WooCommerce → Pickup Store Settings and change the field labels under "Customizable Field Labels".
